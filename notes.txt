list  ( array in C)( mutable)
------------------
list is mutable - elements inside list can be modified
list contains set of elements.
Elments can be set of numbers or strings or  any combination
list is defined in [].

alist = [10,20,30,40]
blist = ["unix","java"]
clist = [10,45.4,"java","oracle",[50,60,70]]



tuple  ( immutable , unchangeable , uneditable )
-------
tuple  contains set of elements.
Elements can be set of numbers or strings or  any combination
tuple is defined in ().
** elements inside tuple cannot be modfied directly ****


# list of lists

dictionary ( other languages : json  hash   hashmap   associativearray    pairs )
-------------------------
dictionary contains keys and values.
key can be number or string to identify the value
value can be number or string or list or tuple or dictionary 
dictionary is defined in {}.

syntax:   object = { key:value , key:value , key:value }
example:   book  = {"chap1":10 ,"chap2":20 ,"chap3":30 ,"chap1":100 }

opening the file 
-----------------

syntax:  object = open(filename,mode)

fobj = open("customers.txt","r") 
# case1 : file should exist to read the file 

fobj = open("customers.txt","w")
#case1 : if the file is not existing .. fil gets created first
#case2 : if the file is already existing.. it overwrites the existing content

fobj = open("customers.txt","a")
#case1 : if the file is not existing .. fil gets created first
#case2 : if the file is already existing.. it appends the new content to the existing


closing hte file
